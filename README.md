# Data Structures and Algorithms

## Overview

This repository contains a collection of data structures and algorithms labs implemented in C. The goal of these projects is to build a strong understanding of core algorithmic concepts, memory management, and performance analysis by implementing everything from scratch.

The labs were done for the COE428 course. Each lab focuses on a different topic, including sorting, recursion, combinatorics, and fundamental data structures.

---

## Labs

- Lab 1 – Insertion Sort and Command-Line Input
- Lab 2 – Towers of Hanoi (Recursion)
- Lab 3 – Sorting Algorithms and Performance Analysis
- Lab 4 – Permutations and State Simulation
- Lab 5 – Stacks and Heaps

---

## Concepts Covered

- Sorting algorithms (Insertion Sort, Merge Sort)
- Recursion and divide-and-conquer
- Time complexity analysis (Big-O)
- Dynamic memory management
- Abstract data structures
- Algorithm performance comparison
